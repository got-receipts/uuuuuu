3. Tap Share.
4. Tap Add to Home Screen.

Vehicle MPG note:

GigOS ships with a local common-vehicle MPG catalog. It does not connect to vehicle manufacturer APIs or gig-platform APIs. Drivers can pick a catalog vehicle or add a custom MPG profile.
